JAVASCRIP

Que es

JavaScrip es un lenguaje de programación que hace interactivas las páginas web al permitir la creación de contenido dinámico, animaciones y efectos visuales, trabajando en conjunto con HTML y CSS.

Cómo funciona

Cuando un usuario visita una página, el código JavaScrip se descarga y es procesado por un motor de JavaScrip que lo traduce de arriba a abajo, permitiendo que se manipulen elementos HTML y CSS sin recargar la página

Para que sirve

JavaScrip sirve para hacer las página web dinámicas e interactivas, permitiendo crear animaciones, validar formularios, actualizar contenido sin recargar la página, y desarrollar aplicaciones web completas, tanto en el lado del cliente (navegador) como en el servidor (backend) usando Node.js.
